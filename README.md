# 🌾 Soil to Shop

### *Direct Farm-to-Market Platform*

---

## 🚀 Overview

Soil to Shop is a smart platform that connects farmers directly with buyers, eliminating middlemen. It ensures fair pricing, better profits for farmers, and lower costs for buyers through transparency and real-time market insights.

---

## ❗ Problem

* Farmers earn less due to middlemen
* Buyers pay higher prices
* No transparency in market pricing
* Inefficient supply chain

---

## 💡 Solution

* Direct farmer-to-buyer connection
* Real-time product listing (crops, vegetables)
* Price insights based on market data
* Easy order and contact system

---

## 🔥 Features

* 🥕 Product Listing (Farmers can add crops)
* 📦 Marketplace (Buyers can view products)
* 🛒 Order Placement
* 📊 Price Insights (Market vs platform pricing)
* 📍 Location-based farmer details

---

## 🏗️ Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite
* **API Testing:** Thunder Client / Postman

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/soil-to-shop.git
cd soil-to-shop
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the server

```bash
python run.py
```

---

## 🌐 API Endpoints

### ➕ Add Product

`POST /products/add`

### 📦 Get Products

`GET /products`

### 🛒 Place Order

`POST /orders/place`

### 📊 Price Insights

`GET /analytics/prices`
